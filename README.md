## 🛠 Development

Clone the repository

```bash
git clone https://github.com/nurodev/nuro.dev.git
```

Install dependencies using npm or yarn

```bash
yarn
```

Start the development server

```bash
yarn dev
```

Build for production

```bash
yarn build
```
